Templates

  • For fast and easy deployments it is possible to use the provided templates.
  • The user just has to fill out the set fields, which are described by a brief statement.
  • For example: Quarkus, Postgres, ...

Namespaces

  • The teachers is able to create endless namespaces for every group project.
  • Be able to work together on the deployment.

Constraints

  • It is important to restrict students to some degree because it is a learning environment.
  • Teachers can set Regex or Min/Max constraints which helps to create a safe place to study.
